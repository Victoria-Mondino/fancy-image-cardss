# Fancy Image Cards

Animated sticky image cards with smooth scroll transitions.

## Preview

Modern stacked image cards effect using:

- HTML
- CSS
- JavaScript

## Features

- Sticky cards
- Smooth scale animation
- Opacity transition
- Responsive layout

## Installation

Clone the repository:

```bash
git clone https://github.com/TU-USUARIO/fancy-image-cards.git
```

Open `index.html` in your browser.
